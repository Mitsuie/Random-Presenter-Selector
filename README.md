# 演習投影学生指名プログラム

本リポジトリは、大学の演習授業などで、課題の投影（発表）を行う学生をランダムに指名し、その実施状況を記録・管理するための一連のプログラムを含んでいます。

## リポジトリに含まれているもの

- プログラム本体（Pythonスクリプト）
  - 名簿変換用プログラム
  - 学生指名用GUIプログラム
- サンプルデータ (`example_files/`)
- 実行ファイル（`.exe`）
  - 名簿変換用プログラム
  - 学生指名用GUIプログラム

## プログラムの構成と使い方

本プログラムは、「名簿の変換」と「学生の指名・記録」の2段階の構成になっています。

### 事前準備
本システムを利用するには、以下のPythonライブラリが必要です。
```bash
pip install openpyxl customtkinter
```

実行ファイルを用いて利用する場合は、外部ライブラリは不要です。

### 使い方（全体の流れ）

1. **名簿の準備と変換**
    使用プログラム：`format_list.py`/ `format_list.exe`
    - 授業ごとに出力されるExcel形式の学生名簿を用意します。
    - その後、変換プログラムを実行し、プログラムで利用できるCSV形式（「投影実施可否」列が追加された状態）に変換して保存します。
    - 授業ごとにこの処理を行ってください。

2. **学生の指名と記録**
    使用プログラム：`select_student_gui.py` / `select_student_gui.exe`
    - プログラムを起動し、1. で作成したCSVファイルを読み込みます。
        - 読み込まないとエラーとなりますので、CSVファイルを都度選択してください。
        - プログラム上部のファイル名表示欄に読み込まれたファイル名が表示されます。
    - 「抽選開始」ボタンを押すと、まだ発表を行っていない（「投影実施可否」が空欄の）学生がランダムに1名選出されます。
        - 選出された学生の学籍番号・氏名・フリガナが画面上部に大きく表示されます。
    - 抽選後、「出席 (○)」「欠席 (×)」などのボタンを押すことで、その結果がCSVファイルに直接記録・保存されます。
        - 「記入なし（戻る）」を押すことで、投影実施可否に情報を記録せず待機画面に戻ることができます。
    - 待機画面に戻った場合は、再度「抽選開始」ボタンを押すことで、次の学生を指名できます。

---

## 各種プログラムの概要

### 1. `format_list.py` (名簿フォーマット変換プログラム)
Excelの学生名簿を読み込み、本システムで利用するCSV形式に変換するスクリプトです。
- 実行するとファイル選択ダイアログが開き、対象のExcelファイルを選択できます。
- 読み込んだデータの末尾に「投影実施可否」という列を自動で追加し、CSVファイルとして任意の場所に保存されます。

### 2. `select_student_gui.py` (演習投影学生指名GUIプログラム)
変換されたCSVファイルを用いて、実際に学生の指名と記録を行うメインのGUIアプリケーションです。
- **ファイル選択機能**: 対象となるCSVファイルをGUI上から選択して読み込みます。
- **抽選機能**: CSVデータのうち、「投影実施可否」が空欄の学生を対象にランダムで1名を選出し、学籍番号・氏名・フリガナを画面に大きく表示します。
- **結果記録機能**: 選出された学生に対して、画面上のボタンで「出席 (○)」「欠席 (×)」を記録できます。結果は即座に元のCSVファイルに上書き保存されます。
